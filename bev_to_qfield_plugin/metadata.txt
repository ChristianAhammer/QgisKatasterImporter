about=
 This plugin converts Austrian cadastral data (BEV - Bureau of Statistics/GIS)
 from MGI/Gauß-Krüger (EPSG:31255) to ETRS89/UTM33N (EPSG:25833) format.
 .
 Features:
  • Batch processing of multiple vector layers (Shapefile, GeoPackage, GeoJSON)
  • Optional NTv2 grid-based coordinate transformation for accuracy
  • Orthometric height calculation using geoid grid
  • Automatic QGIS project creation with styled layers
  • BEV orthofoto (basemap.at) WMTS layer integration
  • QField sync directory preparation for mobile fieldwork
  • Comprehensive processing reports
